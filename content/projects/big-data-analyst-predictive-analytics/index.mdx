Provided a data driven solution for predicting real estate housing prices using AWS SageMaker, Apache Spark and housing data from the years 2012-2018 using various supervised and non-supervised learning models.
Segregated house listings into various market segments using Kmeans clustering on AWS Sagemaker.
Delivered a working model based on random forest algorithm to predict house prices based on housing data from 2012-2018.
